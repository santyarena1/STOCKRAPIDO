# Atajos de teclado – POS

En la pantalla **POS** podés usar el teclado y el escáner para cobrar rápido.

| Tecla | Acción |
|-------|--------|
| **F2** | Poner el foco en la búsqueda (código de barras o nombre) |
| **F4** | Abrir modal de descuento total (monto fijo) |
| **F5** | Cobrar (registrar la venta) |
| **F6** | Pausar venta / Ver ventas en espera (guardar o retomar) |
| **?** | Ver esta ayuda de atajos |
| **ESC** | Cerrar el modal abierto |
| **ENTER** | Con el foco en búsqueda: agregar el primer resultado al carrito |
| **Ctrl + Backspace** | Quitar el último ítem del carrito |

## Modo ráfaga

Escaneá códigos de barras seguidos: cada lectura agrega 1 unidad del producto y el foco sigue en la búsqueda. No hace falta hacer clic entre escaneos.

## Producto manual

Si el producto no tiene código: botón **Producto manual**, ingresá nombre y precio. También se puede usar para ítems sueltos (ej. “Bolsillo”, “Varios”).

## Ventas en espera

- **Pausar (F6)** → Guardar venta actual y abrir otra.
- En el mismo modal podés **Retomar** una venta guardada.

## Descuento

- **F4** o botón “Pausar” en el panel derecho: aplicá un descuento en **monto fijo** sobre el total del carrito.
